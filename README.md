* Familiarity with React

## Description
**Authentication?** I am learning it. **Apollo Data?** Included **Integration with React?** Yes!

This course gets me up and running with GraphQL quickly, and teaches me the core knowledge I need to deeply understand and build React applications quickly.

**Both OSX and Windows supported**

I started by mastering the fundamentals of GraphQL, including data types and schemas, and integration with Express and React. Special attention has been paid to creating code that I am able to make use of on my own projects.

This is the GraphQL course for me, because I am new to GraphQL and have been working to learn it but sometimes feel like I still do not quite get it. I am understanding GraphQL through the followings:
* Learning how to use GraphQL's schema to define relations between my data objects
* Learning the process of combining a backend server with a frontend React
* Mastering the process of thinking about my applications data in terms of a graph structure
* Grasping the difference between GraphQL, Apollo and Relay, and when to use each
* Developing apps that are unique, fun and responsive
* Building servers that can be used with both React and React Native applications
* Mastering handling authentication with GraphQL
* Learning the core principles of navigation with React Router and GraphQL

This is a course that explains the concepts and how they are implemented in the best order for me to learn and deeply understand them.

## Who this course is for
* Engineers looking to learn GraphQL
